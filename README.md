# API de Palavras Aleatórias

Esta é uma aplicação básica desenvolvida em Python utilizando Flask e SQLite. A API retorna uma palavra aleatória de 5 letras a cada requisição.

## Requisitos

- Python 3.x
- Flask
- SQLite
